The ask in one line

Our visitor data sits on a local NAS drive and is hand-shared as Excel files on every marketing request. It is unsafe and scattered. We propose building one secure web application to hold, clean, filter, and act on this data — including sending campaigns through WATI.

01

The Current Situation

Data lives on one local NAS drive

A single drive holds every visitor record — one failure or theft and it is gone.

Excel files shared by hand

Every marketing request means manually exporting and sending a spreadsheet.

Sensitive data spread everywhere

Copies end up in inboxes, laptops, and chats with no control or audit trail.

Data quality keeps degrading

Duplicates, stale entries, and conflicting versions multiply with every export.

02

The Proposed Solution

A single, secure Visitor Data Management web application — one source of truth, accessed through a login, with the tools marketing needs built in.

1
Secure login replaces shared files

Marketing logs in with their own account and views data live — no more emailing spreadsheets.

2
Clean the data in place

De-duplicate, correct, and merge records inside the app whenever required.

3
Filter visitors event by event

Build targeted audiences in seconds by event, date, company, or status.

4
Launch WhatsApp campaigns via WATI

Connect to the WATI API and send campaigns to a filtered segment directly from the app.

03

Key Capabilities

The functional scope proposed for the first release.

Capability What it does
Secure Login & RolesEmail-based sign in with role-based access for marketing and administrators.
Central Visitor RepositoryOne secure source of truth — retires the NAS drive and shared Excel files.
In-App Data CleanupDe-duplicate, merge, and correct visitor records without exporting them.
Event-wise FilteringSegment visitors instantly by event, date, company, and status.
WATI WhatsApp CampaignsSend WhatsApp campaigns to a filtered segment directly through the WATI API.
Controlled ExportExport to CSV only when needed, with every download logged.
Audit Trail & BackupTrack who accessed what, with automatic secure cloud backups.
Overview DashboardLive visitor counts per event and campaign results at a glance.
